# Tugas 1: Aplikasi Kalkulator Web Sederhana (Flask Python)

Project ini dibuat untuk memenuhi Tugas 1 pada mata kuliah Pengantar Pemrograman, Program Studi Teknik Informatika, Universitas Bale Bandung (UNIBBA). Aplikasi ini berupa kalkulator berbasis web sederhana yang dikembangkan menggunakan Python dengan *microframework* Flask, serta dideploy secara online agar dapat diakses oleh publik.

## 🚀 Fitur Aplikasi
* **Operasi Aritmatika Dasar:** Penjumlahan, Pengurangan, Perkalian, dan Pembagian.
* **Responsive Web Design:** Tampilan antarmuka yang bersih dan nyaman diakses baik melalui perangkat desktop maupun mobile.
* **Server-side Validation:** Pemrosesan logika matematika yang aman menggunakan Flask (*backend*).

## 🛠️ Teknologi & Tools yang Digunakan
